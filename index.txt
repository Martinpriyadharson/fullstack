  1.creates the hyperlink to a web page
   
    <a href="webpage link">page name</a>


HTML image tag:
   <img scr="image location" alt="alternative text if the image doesn't load"/>


Image tag + anchor tag:

loading image tag + anchor tag 

FOR EXAMPLE:
   
  1.<img scr="jesusimage jpg"/> <br>
  2.<a href="https://www.youtube.com/watch?.............">christian song</a>

if the above image want to be in the center use the center tag <center>_____</center>

WIDTH Tag:

if we want the image in the bigger size use the width tag:
     
            width="100px"
   
 FOR EXAMPLE:
       
     <img scr="jesusimage jpg"  width="100px>

HTML TABLES AND FORMS:

<table></table>

table has row and column.

<tr>=table row
<td>=table column or table data 


CODE:

<table border="1">
        <tr>
              <td>Name</td> <td>age</td>
        <tr>
        <tr>
              <td>martin</td> <td>19</td>
        <tr>
</table>
